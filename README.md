## Project Overview
This repository contains a complete multi-modal annotated dataset created as a portfolio project to demonstrate professional data annotation skills across three annotation types.

---

## Files in This Repository

| File | Type | Description |
|------|------|-------------|
| `text_annotations.csv` | Text / NLP | 20 e-commerce review sentences labeled for Sentiment & Intent |
| `image_annotations.json` | Image / CV | 10 images with 30 bounding box annotations across 9 object classes |
| `rlhf_evaluation.csv` | RLHF | 10 AI response pairs scored for Accuracy, Clarity, and Helpfulness |
| `Annotation_Report_Adekunle_Feranmi.docx` | Report | Full project documentation including guidelines, QA, and decisions |

---

## Annotation Types Covered

### 1. Text Annotation (Sentiment & Intent)
- **Tool:** Label Studio
- **Items:** 20 sentences
- **Labels:** Positive / Negative / Neutral · Praise / Complaint / Question / Suggestion

### 2. Image Annotation (Bounding Box)
- **Tool:** MakeSense.ai
- **Items:** 10 images, 30 bounding boxes
- **Classes:** car, person, bottle, cup, bowl, chair, bench, laptop, traffic_light

### 3. RLHF Response Evaluation
- **Tool:** Google Sheets
- **Items:** 10 question pairs
- **Scored on:** Accuracy, Clarity, Helpfulness (1–5 scale)
